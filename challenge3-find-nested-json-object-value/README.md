
## Challenge-3 Find nested json object value

## What it does
- Extract value from the nested json object by passing the key.

## Required plugin for pytest
- pip install -r requirements.txt

## How to run
- `python challenge_3.py <dict_object> <key>`
- Run pytest script
  - `pytest --no-header -vv`
